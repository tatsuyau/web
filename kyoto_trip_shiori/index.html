はじめに：京都という舞台

京都の街角には、今もあちこちに「幕末」の傷跡や志が残っています。
このフィールドワークでは、教科書の文字を追うだけでは見えない、
「なぜその場所で事件が起きたのか？」「当時の人々は何を願っていたのか？」
を五感で感じ取ってみましょう。

幕末の歩み：激動の15年

1853

黒船来航

ペリー、浦賀に来航。日本の「鎖国」が終わる。

1863

新選組結成

壬生の地で浪士組が結成される。

1864

池田屋事件

三条木屋町。新選組が長州藩などの尊皇攘夷派を襲撃。

1867

大政奉還・龍馬暗殺

二条城での政権返上。その直後、近江屋で坂本龍馬が散る。

1868

鳥羽・伏見の戦い

戊辰戦争の開戦。旧幕府軍が「賊軍」となってしまう。

LEVEL : 1

壬生：新選組の誕生

壬生寺 / 八木邸 / 前川邸

新選組がかつて「壬生浪（みぶろ）」とよばれ、寺の境内で演習を重ねていた場所。 会津藩の下で京都の治安を守るという使命に燃えた若者たちの「志」を辿ります。

👣 徒歩ナビ：壬生周辺
阪急大宮駅 → (徒歩10分) → 壬生寺 → (1分) → 八木邸・前川邸

🚩 MISSION

LEVEL : 2

三条：政治の転換点

池田屋跡 / 近江屋跡 / 酢屋

鴨川沿いの三条・四条エリア。ここは長州藩や土佐藩、そして坂本龍馬が駆け抜けた「政治の中心地」。 国家のあり方を巡る激しい闘争の跡が、近代的な街並みに隠されています。

👣 徒歩ナビ：三条・河原町
池田屋跡 → (2分) → 酢屋 → (10分 / 約800m) → 近江屋跡

🚩 MISSION

LEVEL : 3

鳥羽伏見：戦火の開戦

城南宮 / 鳥羽離宮跡 / 伏見奉行所跡

1868年。鳥羽の小枝橋で放たれた一発の銃声が、日本の運命を決定づけました。 最新鋭の装備を持つ官軍と、勇猛ながらも旧時代の装備で戦った旧幕府軍。その決定的な違いとは。

👣 徒歩ナビ：伏見・中書島
京阪中書島駅 → (10分) → 寺田屋 → (15分) → 伏見奉行所跡

🚩 MISSION

LEVEL : FINAL

霊山：維新の結果と未来

霊山歴史館 / 龍馬の墓

京都の街を見下ろす東山の高台。ここには明治維新の「勝者」も「敗者」も、共に眠っています。 激動の時代を駆け抜けた人々が、いまの私たちに何を問いかけているのかを考えます。

👣 徒歩ナビ：東山・霊山
京阪清水五条駅 → (徒歩15-20分 / 上り坂) → 霊山歴史館・墓所
